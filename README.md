# Kotlin-Spring-Boot

开发环境规范

开发工具:

        idea

开发版本：
        
        Springboot 版本 2.6.2

        jdk 版本 1.8

        kotlin 版本 1.3.70

添加依赖

Spring Web
````
        <dependency>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-starter-web</artifactId>
        </dependency>
````
Tomcat
````
        <dependency>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-starter-tomcat</artifactId>
                <scope>provided</scope>
        </dependency>
````
        
启动项目
        
        运行KotlinSpringBootApplication.kt的main函数
